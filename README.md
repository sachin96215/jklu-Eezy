## ✨ Features
- 📢 **Announcements** — Campus news & notices  
- 📞 **Contact Directory** — Faculty, staff & admin contacts  
- 🍽️ **Mess Management** — Weekly menu & timings  
- 🔍 **Lost & Found** — Report & recover items  
- 📅 **Class Timetable** — Weekly academic schedule  
- ⚡ **Quick Actions** — Emergency contacts  

## 🛠 Tech Stack
- Flutter Web (Dart)
- GitHub Actions (CI/CD)
- GitHub Pages (Hosting)

## 🏗 Local Development
```bash
flutter pub get
flutter run -d chrome
```

## 📦 Build
```bash
flutter build web --release --base-href "/JKLU-Eezy-Web/"
```
